Features

Four personality quizzes with different themes

Supports single choice, multiple choice, and ranged slider questions

Questions and answers are randomized each time the quiz runs

Timer for each question with automatic skip on timeout

Progress bar showing quiz completion status

Dynamic answer interface using stack views (supports flexible answer counts)

Result calculation based on most frequent selected answers

Local quiz history screen showing past results and completion dates

Navigation flow with introduction screens and return-to-home functionality
